MOOPH = Module for Object Oriented Pseudomultithreaded Hypertext - Example


Two pseudothreads of execution run in pseudoparallel. One of them is a very simple function that adds one to a variable and the other one is a function, that renders that variable and resets its value to zero. Renderer is paused between second and fourth second and adder is paused between fifth and seventh second. Renderer's function is executed every X ms (where X may be set) as measured by MOOPH module and as set by radio buttons. I can not imagine more simple example covering every feature of MOOPH. It is more and more time consuming to render the text as the number of symbols grows...

Renderer's measured delay:  

Set renderer pseudothread to "is measured from last time"
Set renderer pseudothread to "is measured from now"

Set renderer pseudothread to "is measured before call"
Set renderer pseudothread to "is measured after call"
